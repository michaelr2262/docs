# Web Dashboard

<figure><img src="sfx_network.svg" alt="Dashboard" width="64"></figure>

The SentinelFX Dashboard at **[sentinelfx.net](https://sentinelfx.net)** is the web interface for the network. Server moderators and admins can log in to view their server's protection status, and banned users can submit appeals — all without needing Discord open.

---

## Logging In

Visit [sentinelfx.net](https://sentinelfx.net) and click **Login with Discord**. The dashboard uses Discord OAuth — no separate account is needed. Your access level is determined by your roles in the connected server.

{% hint style="info" %}
You must be a moderator or administrator in at least one SentinelFX member server to access the server management section.
{% endhint %}

---

## For Server Moderators & Admins

<figure><img src="sfx_shield.svg" alt="Shield" width="48"></figure>

Once logged in, you can view and manage every server you have staff access to.

### What You Can See

| Section | Description |
|---------|-------------|
| **Overview** | Server name, member count, active modules, and current protection status |
| **Network Bans** | Full list of network-banned users and whether they've been removed from your server |
| **Recent Actions** | Log of recent moderation actions (bans, kicks, mutes, warns) from your server |
| **Appeals** | Incoming appeal requests from users who were banned in your server |
| **Configuration** | View current module toggles, channels, roles, and thresholds |

### Reviewing Appeals

<figure><img src="sfx_appeal.svg" alt="Appeal" width="48"></figure>

When a banned user submits an appeal, it appears in the **Appeals** section of the dashboard. You can review:

- The user's appeal text
- Their full network history and threat score
- The original ban reason and any evidence attached

From there you can set the status to:

- **Under Review** — acknowledge the appeal and let the user know it's being looked at
- **Approve** — grant the appeal and lift the ban across every server simultaneously
- **Deny** — reject the appeal; the ban remains in place

{% hint style="warning" %}
Once an appeal is approved or denied, the status is final and cannot be changed. Take care to review the full history before making a decision.
{% endhint %}

## For Banned Users

<figure><img src="sfx_banned.svg" alt="Banned" width="48"></figure>

If you have an active SentinelFX network ban, log into the dashboard at [sentinelfx.net](https://sentinelfx.net) with your Discord account. If you are eligible to appeal, an **Appeal** button will be shown on your profile page.

Write your appeal (50–1,000 characters), submit it, and the SentinelFX Team will review it. You can check the status of your appeal at any time by returning to the dashboard.

See the full [Appeals](appeals.md) page for guidance on writing a strong appeal.
